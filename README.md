# Resume-Job Matching Using NLP-Based Feature Extraction and Similarity Measurement Techniques

## Project Overview
Repositori ini berisi hasil **_Final Project_ Kelompok 4** untuk mata kuliah **Pengenalan Pola**. Fokus utama _project_ ini adalah implementasi pada dokumen teks untuk menganalisis dan mengukur tingkat kesesuaian antara *resume* dan *job description*. Dengan memanfaatkan teknik **Natural Language Processing (NLP)**, sistem ini mengolah teks tidak terstruktur menjadi representasi numerik, mengekstraksi pola-pola penting dari dokumen, serta menghitung tingkat kemiripan antar dokumen secara sistematis dan objektif.
